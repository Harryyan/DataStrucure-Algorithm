# 二分查找

二分查找法主要是解决在“一堆数中找出指定的数”这类问题。
而想要应用二分查找法，这“一堆数”必须有一下特征：

1.  存储在数组中
2. 有序排列

## 第一个错误版本 (Leetcode-278)


```swift
class Solution : VersionControl {
    func firstBadVersion(_ n: Int) -> Int {
        var left = 1
        var right = n

        while left < right {
            let mid = left + (right - left) / 2

            if isBadVersion(mid) {
                right = mid
            } else {
                left = mid + 1
            }
        }

        return left
    }
}
```
