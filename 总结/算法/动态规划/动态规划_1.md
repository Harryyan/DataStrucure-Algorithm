# 动态规划总结(一)

## 0-1背包变体:

### 分割等和子集 (Leetcode-416)
可以转换为0-1背包问题求解:

```swift
func canPartition(_ nums: [Int]) -> Bool {
    var total = nums.reduce(.zero,+)
    var half = total / 2

    if half * 2 != total {
        return false
    }

    var dp = Array(repeating: false, count: half+1)
    dp[0] = true

    for i in 0..<nums.count {
        for j in stride(from: half, through: nums[i], by: -1) {
            if dp[j-nums[i]] {
                dp[j] = true
            }
        }
    }

    return dp[half]
}
```
